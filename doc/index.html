Notes App

Offline-First • Type-Safe • Production-Ready

A professional-grade notes app built with React Native + Expo to teach real-world app development.

Expo SDK 51+ TypeScript Zod Hermes

Overview

Notes is a fully offline, privacy-first notes application that demonstrates:

  • Clean Architecture (Domain → Data → UI)
  • Type-safe CRUD with zod
  • Secure local storage with AsyncStorage
  • Modern expo-file-system API (File, Paths)
  • Backup/Restore via expo-sharing and DocumentPicker
  • Production builds with code obfuscation

Architecture

src/
├── app/                ← Expo Router screens
├── components/         ← Reusable UI
├── features/
│   ├── folders/        ← Domain logic
│   └── notes/
├── services/backup/    ← exportBackup(), importBackup(uri)
├── utils/storage.ts    ← AsyncStorage + Zod wrapper
└── constants/keys.ts   ← Storage keys

Testing

Troubleshooting

IssueFix
crypto.getRandomValues() errorAdd polyfill in app/_layout.tsx
URI is not absoluteUse Paths.document + new File(dir, name)
Backup file not foundSet copyToCacheDirectory: true in DocumentPicker
